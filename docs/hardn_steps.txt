### **Processes Performed**
1. **System Updates**:
   - Updates all system packages using `apt update` and `apt upgrade`.

2. **Set Hostname**:
   - Sets a generic hostname using `hostnamectl`.

3. **Install Package Dependencies**:
   - Installs essential system utilities and libraries.

4. **Install SELinux**:
   - Installs and configures SELinux in permissive mode.
   - Applies SELinux policies and prepares the system for reboot.

5. **Install Security Tools**:
   - Installs tools like `ufw`, `fail2ban`, `lynis`, and `firejail`.
   - Configures and enables AppArmor profiles.

6. **Install AIDE**:
   - Installs AIDE and initializes its database.
   - Runs an initial AIDE check.

7. **Install Additional Tools**:
   - Installs `chkrootkit` for rootkit detection.
   - Installs Linux Malware Detect (LMD) from its GitHub repository.

8. **Install Rust**:
   - Installs Rust programming language using the official installer.

9. **Apply STIG Hardening**:
   - Configures password policies using `libpam-pwquality`.
   - Locks inactive accounts.
   - Sets login banners for `/etc/issue` and `/etc/issue.net`.
   - Configures kernel parameters for STIG compliance.
   - Secures filesystem permissions.
   - Configures audit rules for critical files.
   - Disables USB storage devices.
   - Disables core dumps.
   - Disables `Ctrl+Alt+Del` key combination.
   - Disables IPv6.
   - Configures `iptables` for strict firewall rules.
   - Enables Address Space Layout Randomization (ASLR).
   - Enforces AppArmor profiles.

10. **Configure Firejail**:
    - Configures Firejail for Firefox and Google Chrome.

11. **Enable Services**:
    - Enables and starts `fail2ban` and AppArmor services.

12. **Firmware Updates**:
    - Installs `fwupd` and applies firmware updates.

13. **Final Setup**:
    - Calls `packages.sh` script for additional setup tasks.
